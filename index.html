Getting started

To execute python code in a Jupyter Notebook the way we did last times, you can follow the following steps:
  1. Open Pycharm and then open the folder where we worked so far.
  2. Open the Pycharm terminal and then type in 
    jupyter-lab
    followed by the ENTER key to launch the notebook.
